*************************************
PrestaShop模块开发手册
*************************************


PrestaShop 是一个非常强大的电子商务平台, 它能够非常容易的扩展一个模块以及重写各种代码.

PrestaShop 基于PHP语言开发，以及JavaScript, HTML, CSS, Smarty模板引擎, SQL与 XML. 它基于MVC构架，此外它还使用了jQuery, Bootstrap, Sass等.

通过这个文档，你可以通过必要的信息来定制你的PrestaShop网站。
